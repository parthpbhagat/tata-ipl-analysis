# 🏏 IPL Data Analytics Power BI Dashboard

## 📌 Project Overview
This project features an interactive Business Intelligence dashboard built using Microsoft Power BI (`ipl.pbix`). It provides an in-depth analysis of historical Indian Premier League (IPL) data, uncovering key trends in team performances, player statistics, and match outcomes through dynamic visualizations.

## 🎯 Key Objectives
As part of an ongoing journey in AI and Data Science, this project demonstrates the practical application of data processing, relational modeling, and advanced DAX queries to derive actionable insights from complex sports datasets.

## 📊 Features & Visualizations
* **Interactive Dashboards:** Dynamic filtering by season, team, and venue for customized insights.
* **Advanced DAX Calculations:** Custom measures developed for calculating Batting Strike Rates, Bowling Economy, and Phase-wise play analysis.
* **Data Modeling:** Optimized Star Schema connecting fact tables (ball-by-ball deliveries, match results) with dimension tables (players, teams) for efficient querying.
* **UI/UX Design:** Clean and intuitive interface for seamless data exploration and storytelling.

## 🚀 How to Use This Project
1. Ensure you have [Microsoft Power BI Desktop](https://powerbi.microsoft.com/desktop/) installed on your system.
2. Download or clone this repository and open the `ipl.pbix` file.
3. Explore the various report pages to interact with the visualizations.

## 🛠️ Technology Stack
* **Tool:** Microsoft Power BI
* **Skills Applied:** Data Visualization, Data Cleansing, DAX (Data Analysis Expressions), Relational Data Modeling

## 🔮 Future Scope

The insights and cleaned data from this Power BI model can serve as a foundation for future machine learning applications, such as predicting match outcomes, forecasting first-innings scores, or clustering player performance types.
